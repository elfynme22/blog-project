# Blog Project

A simple blog app where users can create, edit, view, and delete posts.

## Features
- Create / edit / delete posts (CRUD)
- Post detail page
- Server-side rendering with EJS
- Static assets served from `public/`

## Tech
- Node.js, Express, EJS

## Run locally
```bash
npm install
node app.js
```
Open in your browser: http://localhost:3000
